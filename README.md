# JustWatch Skill

Find where a movie is streaming and play it 

requires a browser plugin and [ovos-media](https://github.com/OpenVoiceOS/ovos-media)

supported platforms include Netflix, Apple TV, Amazon Prime ....

NOTE: these results usually require an account and/or a paid subscription to watch, 
providers either have a flat rate, rent/sell individual movies or are ad supported
